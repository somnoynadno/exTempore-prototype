Обзор

Зачастую работая с информацией и обрабатывая её нам приходится брать эту информацию с различных источников. Многие сервисы предоставляют API (Application Programming Interface), тогда с помощью базовых инструментов мы можем получить в удобном виде информацию. Однако существуют сервисы, которые не предоставляют таких инструментов, а всю информацию содержат в человеко-читаемом формате, к примеру, на сайте.
В таких случаях для автоматизированного получения информации мы вынуждены использовать синтаксический анализ html документа.
В python есть ряд удобных библиотек и инструментов для обработки таких документов. В этом курсе будут рассмотрены некоторые из них.
