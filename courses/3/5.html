Selenium

Библиотека Beautiful Soup хороша для парсинга одного сайта, однако, если ваша задача посетить несколько страниц сайта, возможно, пройти регистрацию, или даже прокрутить страницу для дозагрузки контента, вам понадобятся более сложные сложные инструменты, например с помощью библиотеки requests можно вкладывать headers в запрос, а с помощью консоли разработчика, например, в Chrome вы можете отслеживать запросы javascript, которые выполняются для получения данных.
Однако такой способ может быть весьма трудозатратным. Давайте рассмотрим инструмент selenium.
