Сформулируем тз:
Необходимо получить все ссылки с первой страницы новостей поисковика
Решение:
Разберем это решение:
driver.get('https://google.com') переходим на страницу google.com
driver.execute_script выполняем javascript код
inp = window.document.getElementsByName('q')[0]; находим поле ввода запроса
inp.value = '%s'; вводим значение word в это поле
Дальше находим кнопку и выполняем поиск (кликаем по кнопке)
Наконец находим кнопку "новости" и нажимаем на неё. На последней странице используем поиск по тегу а, чтобы распечатать все ссылки.
